## 📖 遊戲機制與關鍵字定義 (Mechanics & Keywords)
為了確保 AI 與使用者能正確理解卡牌描述，本專案採用以下官方原始定義：
| 關鍵字 | 原始描述定義 |
| :--- | :--- |
| **入場曲** | 從手牌中使用後進入戰場時發動的能力。 |
| **進化時** | 使用 EP 進化，或使用 SEP 超進化時發動的能力。 |
| **超進化時** | 使用 SEP 超進化時發動的能力。 |
| **謝幕曲** | 被破壞時發動的能力。 |
| **疾馳** | 進入戰場的當回合即可進行攻擊。 |
| **突進** | 進入戰場的當回合即可攻擊從者卡。 |
| **守護** | 敵方的從者卡僅可攻擊持有守護能力的從者卡。 |
| **必殺** | 透過交戰給予敵方的從者卡傷害時，將會無視剩餘生命值而破壞該從者卡。 |
| **連擊** | 表示「本回合中自己已使用的卡片張數」。部分能力需在連擊達指定數值以上時可發動。 |
| **倒數** | 自己的回合開始時，倒數將減少 1 回合。當倒數至 0 時，便會被破壞。 |
| **奧義** | 奧義量條為 10 以上時發動的能力（量條 = 目前回合數 + 手牌中此卡見證的進化次數）。 |
| **解放奧義** | 奧義量條為 15 以上時發動的能力（計算方式同奧義）。 |
| **策動** | 1 回合僅限 1 次，透過按下策動按鈕發動。若有標示消費值，則需消耗等同數值之 PP。 |
| **融合** | 在手牌中時，1 回合僅限 1 次，指定手牌中的卡片作為素材並進行融合的能力。 |
| **模式** | 由複數能力中選擇指定數量的能力並發動。 |

> **[給 AI 的特別提醒]**
> 在解析卡片效果文案時，請務必參照上表邏輯。例如當卡片提到「奧義」時，請理解其觸發門檻與回合數、進化次數掛鉤，而非單純的固定數值。
